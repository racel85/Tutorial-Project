JavaScript Can Change HTML Content

One of many JavaScript HTML methods is getElementById().
The example below "finds" an HTML element (with id="demo"), and changes the element content (innerHTML) to "Hello JavaScript":

             
 HTML

             
               <div id="demo">
               Some text
               </div>
               
           
             
CSS

             
             
           
             
JavaScript

             
 document.getElementById("demo").innerHTML =
 "I changed!";

Preview

Some text

Commentary

This will work only if the id is unique.

If you use the same id for multiple elements and try this method, only the first element would change.

JavaScript Can Change Pictures, styles and much more

In this example JavaScript changes the value of the src (source) attribute of an <img> tag:

             
 HTML

             

  <button onclick="engageLight()" type="button" 
class="on-off">ON/OFF<button>

  <img id="bulb" src= "imgs/bulbOFF.png" alt="">

               
           
             
CSS

             
    .on-off{
    width: 50%;
    margin: 2% 25%;
    }

    #bulb {
      width: 30%;
      margin: 0% 35%;
      
    }
             
           
             
JavaScript

             
var state = 0; // the state, 0=OFF & 1=ONN
var bulb = document.getElementById('bulb');

function engageLight() {
  if (state == 0) {
    state = 1; // Register the turning on
    bulb.src = 'imgs/bulbON.png';
    bulb.style.boxShadow =
 "0px 0px 55px 20px rgb(255,215,0)";
    bulb.style.backgroundColor =
 "rgb(255,215,0, 0.72)";

  } else {
    state = 0; // Register the turning off
    bulb.src = 'imgs/bulbOFF.png';
    bulb.style.boxShadow = "0px 0px 0px 0px";
    bulb.style.backgroundColor = "white";
  }
}

Preview

Commentary

In this example, the variable state will tell if the bulb is turned off or on.

A simple if statement will check the state of the bulb and change it.

Javascript can be used to change the scr attribute and any CSS style.
